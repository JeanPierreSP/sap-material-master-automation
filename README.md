# SAP Material Master Automation

Automation of SAP Material Master processes using Python and SAP GUI Scripting.

This project contains scripts designed to automate repetitive SAP tasks related to material master data management, improving efficiency and reducing manual work.

---

## Features

### 1. Extend Material to Storage Location
Automates the extension of existing materials to new storage locations.

Script:
src/extend_storage_location.py

Example input:
examples/example_extend_storage.xlsx

---

### 2. Change Material Description
Updates material short description and long texts automatically.

Script:
src/change_material_description.py

Example input:
examples/example_change_description.xlsx

---

### 3. Batch Scrap Weight Review
Updates batch characteristics such as weight and family classification using MSC2N.

Script:
src/batch_scrap_weight_review.py

Example input:
examples/example_scrap_review.xlsx
